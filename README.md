# UMKM Rejosari Website

UMKM Rejosari Website is a web-based platform designed to provide information and
digital promotion for Micro, Small, and Medium Enterprises (UMKM) located in
Rejosari Village.

This website serves as a digital medium to showcase local UMKM products, increase
business visibility, and help the public easily access information about UMKM
activities in Rejosari Village.
